# Reports Folder

Store controller, worker, QA, and packet reports for this launch-hardening loop.

Naming convention:

```text
C0-baseline-report.md
C1-release-pairing-report.md
C2-feature-flags-report.md
[packet]-[slice]-worker-report.md
[packet]-[slice]-qa-report.md
```

Reports must record exact commands and exact outcomes. Do not claim a check passed unless it was run.
